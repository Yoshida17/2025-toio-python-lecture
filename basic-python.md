# 2. Pythonプログラミングの基礎

## Pythonとは？

Python（パイソン）はとても人気のあるプログラミング言語です．読みやすく書きやすいのが特徴で，ロボットだけでなく人工知能のプログラミングにも適しています．

## 最初のプログラム

次のプログラムをThonnyのエディタに入力してみよう．

```python
print("こんにちは！")
print("Pythonプログラミングを始めよう！")
```

このプログラムを実行すると，シェル画面に「こんにちは！」と「Pythonプログラミングを始めよう！」が表示されます．

### プログラムの実行方法

1. Raspberry Piのデスクトップから「Thonny」を起動する
2. エディタ画面に上記のコードを入力する
3. `Ctrl + S`または「Save」ボタンをクリックしてファイルを保存する（保存ファイル名の例：`hello.py`）
4. `F5`キーまたは「Run」ボタンをクリックしてプログラムを実行する

### ポイント

- `print()`関数は，括弧の中に書かれた文字列を画面に表示する
- 文字列は必ずダブルクォーテーション`"`で囲む
- 日本語入力，英数入力の切り替えは`Ctrl + Space`またはキーボード左上の`半角/全角`キーで行う
- Pythonでは大文字と小文字が区別されるため，`Print`や`PRINT`ではエラーになる

## コメントアウト

プログラムの中に説明文を書く方法です．`#`から行末まではプログラムとして実行されません．

```python
# これはコメントです。実行されません
print("この行は実行されます")  # この部分はコメントです

# 複数行のコメント
# このように書くと
# 複数行にわたって
# コメントを書くことができます

"""
三重引用符で囲むと
複数行のコメントを
一度に書くことができます
"""
```

コメントはプログラムの説明や，一時的にコードを無効にするために使います．

## 変数とデータ型

変数はデータを保存するための「箱」です．`=`は代入演算子と呼ばれ，右側の値を左側の変数に代入（保存）する意味を持ちます．

```python
# 変数に値を入れる
name = "Tanaka"        # 文字列型（str）- 文字や文章を保存
age = 13               # 整数型（int）- 整数を保存
height = 152.5         # 浮動小数点型（float）- 小数を保存
is_student = True      # 論理型（bool）- TrueまたはFalseを保存

# 変数の内容を表示する
print("私の名前は", name, "です")
print("私は", age, "歳です")
print("身長は", height, "cmです")
print("生徒ですか？", is_student)
```

### 主なデータ型

- **文字列型（str）**: `"Hello"`，`"こんにちは"`など，文字や文章を扱う
- **整数型（int）**: `10`，`-5`，`0`など，整数を扱う
- **浮動小数点型（float）**: `3.14`，`-2.5`など，小数を扱う
- **論理型（bool）**: `True`または`False`の2つの値のみ

### 変数名のルール

1. 英数字とアンダースコア（`_`）のみ使用可能
2. 数字から始めることはできない
3. 大文字と小文字は区別される
4. Pythonの予約語は使用できない

```python
# 正しい変数名の例
student_name = "Sato"
score1 = 85
MAX_VALUE = 100

# 間違った変数名の例（エラーになる）
# 1st_name = "Taro"    # 数字から始まっている
# class = "A組"        # Pythonの予約語
```

## 演算子

次の演算子を使うと計算をすることができます。

```python
# 足し算、引き算、掛け算、割り算
a = 10
b = 3

print("a + b =", a + b)  # 足し算
print("a - b =", a - b)  # 引き算
print("a * b =", a * b)  # 掛け算
print("a / b =", a / b)  # 割り算（小数点あり）
print("a // b =", a // b)  # 割り算（整数部分のみ）
print("a % b =", a % b)  # 余り
```

## インデント（字下げ）の重要性

Pythonでは**インデント**（行頭の空白）がとても重要です。インデントが正しくないとプログラムが動きません。

行頭にカーソルがある状態で`Tab`キーを押すとインデントが追加されます。

```python
# 正しいインデント
if 10 > 5:
    print("10は5より大きいです")  # インデントされている
    print("これも同じブロックの中です")  # 同じ量のインデント

# インデントがなければエラーになります
if 10 > 5:
print("これはエラーになります")  # インデントがない！
```
